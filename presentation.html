The Solution

Six powerful modules.
One futuristic dashboard.

Service Business Hub replaces spreadsheets with a real-time AI-assisted workspace — designed for the way service professionals actually work.

📊
Live Dashboard
Real-time stats: total clients, overdue follow-ups, draft messages, scheduled content — with contextual alerts.
👥
Client Manager
Full CRM with contact details, service type, tags, last-contact tracking, message history and follow-up log per client.
✨
AI Outreach Composer
Generate personalised re-engagement, check-in, upsell, and referral messages. Saves draft + updates last_contact_date automatically.
🔔
Follow-Up System
Set reminders per client. Urgency levels: Overdue / Due Today / Upcoming / Snoozed. Alerts for 30-day inactive clients.
📅
Content Calendar
AI-generates LinkedIn, Instagram, Blog & Newsletter posts from a topic + niche input. Schedule with list or weekly calendar view.
📚
Content Library
Save & reuse best-performing templates. Track use counts, edit inline, filter by category. Sorted by performance automatically.

Design Decisions

Glass-morphism over flat cards
rgba(255,255,255,0.03) backgrounds + backdrop-filter: blur(20px) creates depth without heavy visuals. Every card feels like frosted glass in a neon-lit space.
Inline style for glow effects
Arbitrary Tailwind shadows can be inconsistent across build outputs. All neon glows use inline style={{ boxShadow: "..." }} for zero-risk rendering.
Consistent motion language
hover:scale-[1.02], hover:-translate-y-0.5, and transition-all duration-200/300 across all interactive elements. Feels alive without being distracting.
rgba badges instead of Tailwind colour classes
Status badges use inline rgba for both bg and border — fully legible on the dark bg without any dark-mode CSS variables.
